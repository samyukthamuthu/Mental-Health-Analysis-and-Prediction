# Mental Health Analysis and Depression Prediction

This project focuses on predicting depression using Machine Learning based on mental health and lifestyle factors such as academic pressure, work pressure, sleep duration, financial stress, study satisfaction, job satisfaction, and family history of mental illness.

The model was trained using Python and Scikit-learn and deployed using Streamlit to create an interactive web application.

## Live App

https://samyukthamuthu-mental-health-analysis-and-prediction-app-tnlpvvu.streamlit.app

## Features

- Machine Learning prediction using Random Forest
- User-friendly interface using Streamlit
- Input using words instead of numeric codes
- Real-time prediction
- GitHub + Streamlit deployment

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## How to Run Locally

```
pip install -r requirements.txt
streamlit run app.py
```

- Not Depressed

and provides an opinion about mental health condition.
